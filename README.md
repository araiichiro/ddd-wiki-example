# ddd-wiki-example
Domain Driven Design simple example (Wiki application)

## Install

```sh
$ git clone https://github.com/signdoubt/ddd-wiki-example.git
$ cd ddd-wiki-example
$ pip install -e .
```

## Launch

```sh
$ mywiki
```

Then, access to http://localhost:5000/
